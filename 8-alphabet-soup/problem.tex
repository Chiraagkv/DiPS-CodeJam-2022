  \section*{Prompt} 
  Every string has an ``alphabet'',composed of the characters that make it up. For example, the alphabet of $aabbccabcabc$ is $a, b, c$. There are two operations you can do with alphabets: getting the alphabet of a string, and seeing if another string has a given alphabet. Given two strings, you must write a program that finds the alphabet of the first string, and returns \textit{true} or \textit{false} based on whether that alphabet makes up the second string.
  \subsection*{Input Format}
    The first and only line of input contains 2 space-separated strings which you have to evaluate.
  \subsection*{Output Format}
    The first and only line of your output must contain \texttt{true} or \texttt{false}, depending on the test case.
  \subsection*{Constraints}
    $10 \le \text{length of strings} \le 100$
