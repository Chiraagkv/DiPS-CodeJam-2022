  \section*{Prompt}
  Given an integer $n$ and an array of terms that are $\le n$, find the minimum number of numbers from the array that sum up to exactly $n$.

  \subsection*{Input Format}
  \begin{itemize}
    \item The first line of input contains the integer $n$.
    \item The next line of input contains an array of terms that are $\le n$.
  \end{itemize}
  \subsection*{Output Format}
  The first and only line of your output must contain the minimum number of numbers from the array that sum up to exactly $n$.
  \subsection*{Constraints}
  \begin{itemize}
    \item $ 2 \le n \le 100 $
  \end{itemize}

    \hline

  \section*{Solution}
   To find the minimum number of numbers from the array that sum up to exactly $n$, take the following steps:
   \begin{itemize}
    \item First, find all the permutations of numbers from the list that sum up to $n$.
    \item Second, Find the smallest number of numbers that can bee added to produce $n$. This can be achieved by sorting the list of permutations by the number of numbers in each permutation.
   \end{itemize}
